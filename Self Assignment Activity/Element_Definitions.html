Chapter 2 Definitions:

  • h1 - a sub-header used in the body of a webpage.
  • hr - is a horizontal line that is used to seperate paragraphs/text in the body of the webpage.
  • br - A break line is used to create a space in between text throughout the webpage.
  • p - This stands for paragraph and is used for text on the webpage.
  • b - This creates the word to be bold.
  • i - This creates the word to be italicized.
  • sup - This is used to create symbols that appear above the word. These are normally used for footnotes.
  • sub - This is used to create symbols that appear below the word. These are normally used for chemical formulas.

Chapter 3 Definitions:

  • Nested Lists - A ul tag inside of another ul to create a sub bullet.
  • Ordered Lists - Orders text by numbers.
  • Unordered Lists - Orders text with bullets.
  • dl - Defines a description list.
  • dt - Describes a term/name in the description list.
  • dd - Describes a term/name in the description list. In this tag you can input paragraphs, images, links, line breaks, etc.
